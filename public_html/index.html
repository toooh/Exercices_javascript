Nous allons faire tous nos exercices de Javascript via ce projet.
Le but de ce projet est de réviser les principes de versionning, d'écrire nos premières lignes de JS et surtout d'aller fouiller dans du code.

Le but est aussi de faire un projet à votre image. Si vous souhaitez que le fond soit rose, n'hésitez pas.
Les seules consignes à respecter sont :
  • Laisser le fond des exercices en l'état. Vous pouvez modifier la forme telle que les photos, css..
  • Commiter sur votre repository à la fin de chaque cours de Javascript même si l'exercice n'est pas fini.
  • Finir l'exercice en cours chez vous si vous n'avez pas eu le temps de le finir dans le temps imparti. Ces exercices constitueront une partie de la note de TP.
