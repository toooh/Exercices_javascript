LES TABLEAUX

Réaliser le pseudo-code suivant :
  • Parcourez un tableau de nombres.
  • Affichez les valeurs les unes en dessous des autres précédées d'un "+" sauf pour le premier élément.
  • Additionner tous les nombres.
  • Lorsque vous avez fini de parcourir le tableau,
    affichez le résultat de l'addition sous une barre afin de représenter l'addition posée.
